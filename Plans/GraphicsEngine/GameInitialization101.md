## A simple initialization algorithm

This stuff is a part of game startup
Game startup is fully platform-dependent stuff. For each platform it's unique.

This is the list from _GameCodingComplete_ book

1. Check system resources: hard drive space, memory, input and output devices
1. Check the CPU speed
1. Initialize your main random number generator (this was covered in Chapter 3).
1. Load programmer’s options for debugging purposes
1. Initialize your memory cache
1. Create your window
1. Initialize the audio system
1. Load the player’s game options and saved game files
1. Create your drawing surface
